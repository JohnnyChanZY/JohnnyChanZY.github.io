# 欢迎来到我的博客

这是我使用 Markdown 撰写的第一篇技术博客文章。

## 关于这个博客

这个博客基于纯静态 HTML/CSS/JavaScript 构建，使用 Markdown 作为文章格式，通过 marked.js 在前端解析渲染。

## 未来计划

我计划在这里分享：

- **技术教程**：Web 开发、人工智能等领域的学习笔记
- **项目经验**：实际项目中的踩坑与解决方案
- **思考随笔**：关于技术与职业的个人见解

## 技术栈

- 静态站点托管：GitHub Pages
- 样式：纯 CSS，无框架
- Markdown 解析：[marked.js](https://marked.js.org/)
