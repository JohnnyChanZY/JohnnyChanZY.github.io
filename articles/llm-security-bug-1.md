# GPT-4 发现 Linux 内核内存泄漏漏洞

在最近的安全研究中，GPT-4 通过分析 Linux 内核源码，成功发现了一个存在于网络子系统的内存泄漏漏洞。

## 漏洞详情

- **影响组件**: `net/core/skbuff.c`
- **漏洞类型**: 内存泄漏 (CWE-401)
- **发现方式**: AI 辅助代码审计

研究人员使用 GPT-4 对内核代码进行大规模静态分析，模型准确指出了 `skb_copy_expand()` 函数中的异常处理路径缺少 `kfree_skb()` 调用。
