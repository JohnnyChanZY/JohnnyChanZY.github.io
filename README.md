# JohnnyChan 个人博客

一个几何风格的个人博客网站。

## 预览

![Blog Preview](preview.png)

## 特性

- 🎨 几何美学设计
- ✨ 流畅的动画效果
- 📱 响应式布局
- 🌙 深色主题
- ⚡ 轻量快速

## 本地预览

```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx serve .
```

然后访问 http://localhost:8000

## 部署到 GitHub Pages

1. 将文件推送到 `JohnnyChanZY.github.io` 仓库
2. 访问 https://JohnnyChanZY.github.io

## 自定义

修改 `index.html` 中的内容来更新：
- 个人简介
- 博客文章
- 社交链接

## 技术栈

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript
- Google Fonts (Space Grotesk, Noto Sans SC)
